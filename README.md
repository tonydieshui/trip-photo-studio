# 旅图整理台

一个本地优先的旅行照片素材管理桌面应用。它可以引用任意本地素材文件夹，在不移动、不修改原片的前提下完成缩略图浏览、基础质量提示、快速筛选和精选导出。

## 当前版本

- 从任意本地文件夹及其子文件夹导入照片
- 支持 JPEG、PNG、WebP、GIF、BMP、AVIF、TIFF、HEIC 及常见 RAW 文件的素材登记
- 后台生成并缓存缩略图
- 本地分析曝光、清晰度、对比度、饱和度、色温、动态范围和明暗剪切
- 生成照片主色板、横竖幅、视觉重心、构图倾向、风格标签和一句话摘要
- 汇总整组照片的平均技术分、常见风格、色调倾向和项目色板
- 按暖色、冷色、低饱和、明亮、暗调、极简等视觉特征筛选
- 通过“颜色实验室”按红橙黄绿青蓝紫粉等主色，以及互补色、邻近色、单色系、莫兰迪、柔和粉彩、大地色、青橙电影感、日落色、森系等条件筛选
- 以“未筛选、保留、待定、淘汰”和 0～5 星管理照片
- 全屏看图和键盘快速筛选
- 将所有“保留”照片复制到新的精选文件夹
- 筛选记录保存在应用本地，原片始终保持不变

RAW、HEIC 的缩略图能力取决于系统是否提供对应解码器；即使不能预览，也可以正常标记并导出原文件。

## 本地运行

```powershell
pnpm install
pnpm start
```

在当前 Windows 工作区中可以直接双击 `启动旅图整理台.cmd`。如果系统仍对中文文件名处理异常，也可以双击内容完全相同的 `launch.cmd`。

## 键盘快捷键

- 左右方向键：上一张 / 下一张
- `P`：保留
- `U`：待定
- `X`：淘汰
- `0`～`5`：星级
- `+` / `-`：大图放大 / 缩小
- `F`：恢复“显示完整”
- 鼠标滚轮：大图连续缩放
- 放大后按住图片拖动：查看局部
- `Esc`：退出大图

## 数据安全

第一版采用“引用导入”：项目记录的是原照片位置，应用不会移动、重命名、覆盖或删除原文件。导出时只会把“保留”的文件复制到用户选择的新目录。

## 关于照片理解

当前视觉理解完全在本机运行，不上传照片。技术分只评价曝光、层次、明暗剪切和细节状态；色调、构图和风格只作为可解释标签展示，不会替用户决定照片是否值得保留。当前版本不包含人物、食物、建筑等对象识别模型。

颜色筛选基于缩略图色板的色相、饱和度、明度及色相环角度关系近似判断。互补色通常要求两组明显颜色在色相环上相距约 150°～210°；莫兰迪色系则综合低至中等饱和度、灰度感与适中的明度判断。它适合快速发现候选素材，但不是严格的艺术风格认证。
